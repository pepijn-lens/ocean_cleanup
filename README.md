# Ocean Cleanup Challenge Solver

A beam search solver for The Ocean Cleanup Challenge optimization problem.

## Requirements

- Python 3.x
- `openpyxl` library

## Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the solver:

```bash
python solver.py
```

The solver reads `toc-challenge-data.xlsx` and outputs the optimal route with rotation codes for each day.
